CERES Overview Dashboard — Redesign Spec

Approved design decisions for review before implementation

Design Principles
🎯
No Useless Data
Every number, table, and alert must lead to a decision or action. If it can't, delete it.
👆
Every Alert Has a CTA
Problems without solutions are noise. Every alert maps to an API action.
📊
Intelligence First
Rate data and quality metrics get the most real estate. Health is a glanceable bar.
Core Decisions
Audience
Hybrid — Health + Intelligence
Product/business focus with ops health as a glanceable status bar. Rate intelligence gets primary real estate.
Layout
Dashboard Zones
Thin health bar → KPI cards → Rate Intelligence table (65%) + Right sidebar (35%)
Data Quality
New — First-class metric
completeness_score and data_confidence surfaced in KPI card, table column, and quality summary panel.
Alerts
Actionable Only
5 alert categories, each with a CTA button that triggers an API action. No passive alerts.
Page Zones (Top → Bottom)
1
Health Status Bar
Thin strip at top. Bank status counts (colored dots), live pipeline progress, 7d success rate with trend. Always visible, never dominates.
● 40 active ● 13 unreachable ● 5 blocked Pipeline: Crawler ▶ 34/58 64% success ↓8%
2
KPI Cards (4 across)
Each card: big number + delta + 7-day sparkline. All numbers have context — trends, comparisons, direction.
Banks Monitored (58, +3) Loan Programs (288, +12 new) Avg KPR Rate (7.2%, ↓0.3%) Data Quality (72% avg completeness)
3
Rate Intelligence Table (~65% width)
Main content area. Tabbed by loan type. Rich columns: bank + status dot, min/max rate (color-coded), 7d trend chip, completeness bar, confidence dots. Replaces the broken heatmap.
Tabs: KPR | KPA | Multiguna | Kendaraan | All Color: <6% green, 6-8% yellow, >8% red Sortable columns Click row → Bank detail
4
Right Sidebar (~35% width)
Three stacked panels: Needs Attention (actionable alerts with CTAs), Today's Changes (meaningful deltas), Data Quality Summary (high/medium/low distribution).

What's Removed
Deleted from Overview
❌ Live Activity feed — was 90% "system ping" noise, replaced by Today's Changes
❌ Old Rate Heatmap — showed all dashes, replaced by tabbed Rate Intelligence table
❌ Ring chart for success rate — moved to health bar as inline text with trend
❌ Sidebar crawl progress panel — consolidated into health bar pipeline status
New Backend Requirements
EndpointPurposeData Source
GET /api/dashboard/alerts Aggregated alerts across all 5 categories crawl_logs, loan_programs, banks, bank_strategies
